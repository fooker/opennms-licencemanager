Easy digital Downloads plugin to enable selling of OSGi Licencces

== Description ==

This plugin extends the Easy digital Downloads plugin to allow it to inteact with the OSGi Licence manager
running in a Karaf instance.

Thanks to Hugh Lashbrooke for providing the WordPress Plugin Template on which this is based
https://github.com/hlashbrooke/WordPress-Plugin-Template

== Installation ==

Installing "eddOsgiLicences" can be done either by searching for "eddOsgiLicences" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is the plugin for? =

The OSGi licence manager controls which OSGi modules start up in a KARAF container.
This plugin allows programmers to create and sell licences for use with the manager
